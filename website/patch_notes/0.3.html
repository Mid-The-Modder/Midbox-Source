Midbox Revolution

Midbox Revolution, or Midbox V0.3, is the largest update to come to Midbox yet.
This update syncs with JummBox, takes various things from other Beepmods, and adds many unique things.

New Content

Presets

‣ Added 8 preset categories.
The new categories are: Midbox Drum Presets, Midbox Piano Presets, Midbox Lead Presets, Midbox FM Presets, Midbox Wavetable Presets, Midbox SFX Presets, Midbox Bass Presets, and Midbox Synth Presets.

‣ Added, ported, renamed, changed, and reorganized all of the presets.

New Instrument Types

‣ The Supersaw from BeepBox V4.2!
An instrument type that uses many sawtooth/pulse width waves that shift in frequency using adjustable chorus-like settings.
Comes with modulators for each of its settings.
‣ Advanced FM (ADVFM) from GoldBox!
This instrument is a more complex counterpart to the regular FM, using 6 operators instead of 4 and customizable algorithms/feedbacks.
Comes with a few extra modulators and envelopes for it's new sliders.

‣ The Wavetable!
An instrument type that one-ups Sandbox's duty cycle instrument type. It uses 32 custom chips in a cycle where they swap through each other with adjustable speed.
Comes with 2 modulators to control the speed and set the current wave of the cycle.
The wavetable also has an interpolation checkbox underneath its speed slider that allows the waves to smoothly transition between eachother, as well as two more checkboxes for if the wavetable should reset the cycle per note, and if so, the other for making the cycle end at the final wave.

New Effects

‣ Wavefold!
An effect for making an instrument's waveform invert direction when hitting certain boundaries.

‣ Clipper!
An effect that makes the instrument's sound clip when it's too loud.

Effects | Chord Type

‣ Added the strum speed slider, as well as its modulator.

‣ Added 'arpeggio patterns' for arpeggio chords.
It is a built-in pattern system which the arpeggio will use to cycle through notes with in an arpeggio chord.

Effects | Transition Type

‣ Added the slide speed slider, as well as its modulator.

‣ Added a checkbox that toggles whether a seamless transition is cut off at the end of a pattern.

Keybinds

‣ Added 7 keybinds.
These keybinds are B, X, `, CTRL+CLICK, CTRL+R, Shift+F, and Shift+ENTER.
|
| The X keybind "cuts" the pattern, removing all notes from it while also copying them. (ModBox)
| The B keybind now activates a temporary loop bar mode, while the 'beats per bar' prompt keybind was moved to 'SHIFT+B'. (JummBox & UltraBox)
| The ` keybind opens the 'song recovery' prompt.
| Holding CTRL and drag clicking on a filter/slider will record the value to a modulator as the song plays. (JummBox)
| CTRL+R takes you to the new random generation settings prompt.
| Shift+F moves the playhead to the start of the song's loop. (Unbox)
| Shift+ENTER adds a new bar(s) behind the selection. (Voxbox)

‣ Finished the ~ (SHIFT+`) keybind for starting a new blank song.

‣ Added a prompt in the preferences menu that allows toggling keybinds on/off called 'Keybind Setup...'.

‣ Added the P keybind to the song player to quickly go back to the editor.

‣ Made the R and Shift+R keybinds (randomizes instrument) work with selections.

Unisons

‣ Added unison compatibility to the pulse width, spectrum, and noise instrument/drum types.

‣ Added 2 unisons.
They are Bent and Hecking Gosh (GoldBox).

Chipwaves

‣ Added 6 chipwaves.
These chipwaves are Accurate Sine, Accurate Triangle, Accurate Trapezoid, Secant, Trapezoid (GoldBox), and Glitch 2.

Envelopes

‣ Added 10 envelopes.
The new envelopes are: Linear, Rise, Wibble, Mini Tremolo, Full Tripolo, Semi Tripolo, Mini Tripolo, JummBox Blip, LFO, and Custom (Basic).
The Linear/Rise/Wibble envelopes were ported from GoldBox. JummBox Blip was ported from JummBox V2.6. LFO and Custom (Basic) are envelopes unique to Midbox.

‣ Added a dropdown containing more envelope settings. (JummBox Addition)
The only setting in it is a instrument-wide envelope speed.

‣ Added envelope targets for distortion, bit-crusher, freq-crusher, chorus, and reverb.

‣ Added a dropdown per envelope with settings for customizing their speed individually, envelope stretching boundaries, whether each one is discrete individually, as well as many other fun settings.
There is also an envelope graph which displays the envelope at an adjustable time length for viewing envelope behaviors as settings are changed.

‣ Ported Slarmoo's Box's 'pitch' envelope type with various new settings to experiment with. Not compatible with drumset envelopes.

‣ Restructured how envelopes look with various new buttons and a new envelope layout.
Some of these new buttons include buttons which reorder, copy/paste, and randomize envelopes, with even more buttons to play with.

‣ Most of these changes have been ported to the drumset for consistency.

FM-Centric

‣ Added 3 FM operator frequencies.
These are 48×, 64×, and 128×.

‣ Added 11 FM operator waveforms, including waves from the YMF262 FM Synthesizer!
These are Clang, Metallic, White Noise, Secant, Quasi-Sine (UltraBox), Absine (YMF262 Absolute Sine), Semisine (YMF262 Half Sine), Camelsine (YMF262), Pulsine (YMF262 Alternating Sine), Sharksine (YMF262 Psuedo-Sawtooth), and Logarithmic Saw (YMF262).

‣ Remade the FM pulse width to act like a normal pulse width, as well as giving it modulation and envelope support. (UltraBox Addition)

‣ Added many more algorithm and feedback options to the FM.

Custom Chip-Centric

‣ Added copy/paste wave buttons in the custom chip's UI and prompt.

‣ Added drawing tools to the custom chip's prompt.
There are 4 tools/modes; Cursor, Line, Curve, and Selection.
- Cursor is the default draw mode where you click to place a box representing one of the custom chip's samples.
- The line tool draws a straight line from where you first clicked to where you release it.
- The curve tool first draws a line on the first drag click, and then on the second drag click, bends that line based on the mouse position.
- The selection tool creates a box, which you can drag and use keybinds in to change a specified part of the custom chip.
‣ Added the 'load preset' button into the custom chip prompt.

‣ Added a randomization button in the custom chip prompt which randomizes the waveform.

‣ Various keybinds now work in the custom chip prompt. E.g: + and - shift wave samples up/down. R randomizes waveform. C and V copies/pastes the waveform.

Basic Noise-Centric

‣ Made basic noise both an instrument type and drum type. (Dogebox2)

‣ Ported 12 noise types from UltraBox, GoldBox, ModBox, and Sandbox.
The new noise types are: Static, Retro Clang, Harsh, Trill, Detuned Periodic, Snare, Chime, 1-Bit White, 1-Bit Metallic, Crackling, Pink, and Brownian.

‣ Added 4 new noise types.
The new noise types are: Perlin, Fractal, Vinyl Crackle, and Noise Square.

Randomly Generated Instruments

‣ Added all features to the random generator.

‣ Added the custom chip and drumset into the list of possible randomly generated instruments.

‣ Added a prompt for better control on how your instrument randomly generates.
It can be found in the edit menu or by using the CTRL+R keybind.

UI Changes and Additions

‣ Dropdown items now fade in when the dropdown button is pressed. These items are marked with "‣ ". (JummBox Addition)

‣ Added speed displays next to all speed sliders. (JummBox Addition)
They can be measured in ticks, multipliers, or waves per beat (wavetable-only case).
The affected sliders are: Arpeggio Speed (multiplier), Vibrato Speed (multiplier), Strum Speed (ticks), Slide Speed (ticks), Envelope Speed (multiplier), and Wavetable Speed (WPB).

‣ Add a new 'knob' UI element!
Used when possible to shorten space on setting groups.

Themes

‣ Added the Blutonium theme. (JummBox Addition)

‣ Added color variables for the oscilloscope lines and mod name.

Misc

‣ Added random splash text under the mod's name.

‣ Added the song subtitle right under the song title.

‣ The song title, subtitle, and channel names can now go up to 63 characters instead of 15 or 30. (Unbox Change)

‣ The channel name can now be changed directly from its spot in the editor.

‣ Added a new mod logo.
The old mod logo was a pink version of JummBox's. The new one is a little pair of headphones.

‣ Ported the "song key octave" box from UltraBox.

‣ Ported GoldBox's oscilloscope, as well as a preference to disable it and a slider that scales the waveform by the Y axis.

‣ Doubled the amount of options on the distortion slider.

‣ Put a few more options on the fade in/out editor.

‣ Added picked string 'sustain types' in the 'sustain' tip prompt. (Unfinished BeepBox Addition)
You can choose between acoustic (A) and bright (B) sustain.

‣ Added a "bass offset" option in the note recording prompt for two-hand keyboard playing. (JummBox Addition)

‣ Added a 'custom' scale that can be edited for making your own scale. (GoldBox Addition)

‣ Added import modes for importing songs from other mods correctly. (UltraBox Addition)

‣ Added an "advanced settings" dropdown underneath general instrument settings for changing more complex niches about the instrument, mainly for convenience.

Bug Fixing

‣ Fixed an issue where vibrato/arpeggios would desync when exporting the song to a sound file.

‣ Made the detune envelope work like the pitch shift's for usability. (JummBox Bugfix)

‣ Fixed an issue where the ModBox Bow envelope had negative values and when used on a pulse width, would make a loud click sound.

‣ Fixed an issue where if the theme was "BeepBox Competition" the fourth noise channel's notes would appear black.

‣ Fixed an issue where spectrums would sometimes sound irregular until edited.

‣ Fixed an issue where if the "keep current pattern selected" preference was enabled, editing the custom chip while the pattern changed would reset what was drawn on the custom chip.

‣ Fixed drawing on custom chip for mobile users.

‣ Fixed an issue where users could not type more than 10 into filter mod point coordinate modulators. (Slarmoo's Box Bugfix)

‣ Disabled randomization keybinds on modulator channels; fixes an issue where pressing "Shift+R" on an active modulation channel would corrupt the song.

‣ Fixed an issue where newly added mod channels were not added underneath the currently selected channel.

‣ Fixed an issue where pitch shift couldn't be added from the modulator select when not on the instrument.

‣ Fixed an error with the track editor's height. (UltraBox Bugfix)

Removed

‣ Strum-Based Chord Types

This refers to the various strum-like chords that varied in speeds.
They were removed due to not being needed anymore upon the addition of strum speed.
[REMOVED]: Medium Strum, Slow Strum, ⅓ Strum, ⅓ Medium Strum, ⅓ Slow Strum, and Build Up chord types.

‣ 'In Pattern' Transition Types

This refers to the transition types that would be cut off at the end of a pattern.
They were removed due to not being needed anymore upon the addition of the 'continue through pattern/in pattern' checkbox.
[REMOVED]: Continue in Pattern and Slide in Pattern transition types.

‣ Insta-Slide Transition Type

This refers to the transition type that was a very fast version of slide.
It was removed due to not being needed anymore upon the addition of slide speed.
[REMOVED]: Insta-Slide transition type.

‣ Many Envelopes

The list of envelopes has now changed with the introduction of the envelope system overhaul.
The naming scheme of "Type SpeedNumber" was removed since indiviudal envelope speed takes that placement smoothly, as well as the fact that the envelope list was becoming much too large, and thus needed to be cut down to allow for more unique envelopes.
[REMOVED]: Envelope speeds named by numbers, tremolo variants indicated by numbers, long envelope list.

‣ The Localization System

The language prompt and the option to change between manually-translated languages has been removed.
It's too much work to be worth it. It also made programming some of the features in annoying due to the constant lack of words when it comes to translation, and figuring that all the other languages would be much more difficult or just straight up not have any available words to describe things, it's not worth the effort.
[REMOVED]: Language Select Prompt.
